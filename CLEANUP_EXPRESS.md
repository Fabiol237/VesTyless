# 🚀 NETTOYAGE EXPRESS - 2 MINUTES

## Option 1: SQL (Plus Simple) ✨

### Étape 1
Va à: **Supabase Dashboard → SQL Editor**

### Étape 2
Copie **TOUT** le contenu de `supabase/CLEANUP_FINAL.sql`

### Étape 3
Colle dans l'éditeur SQL et clique **Run**

### Résultat
- Vieilles boutiques vides → désactivées ✅
- Tes vraies boutiques → réactivées ✅
- Produits cassés → corrigés ✅

**C'est fait en 30 secondes!**
