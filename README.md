## Shipday Integration (Real-time Tracking)

To enable Shipday order creation and customer live tracking, set these environment variables:

```bash
SHIPDAY_API_KEY=your_shipday_api_key
SHIPDAY_DEFAULT_PICKUP_ADDRESS=Your pickup address if store address is missing
SHIPDAY_DEFAULT_PICKUP_PHONE=+2250000000000
NEXT_PUBLIC_APP_URL=http://localhost:3000
```

Notes:

- Keep `SHIPDAY_API_KEY` on the server only. Do not expose it in client-side variables.
- Tracking page URL format is `/track/{trackingId}`.
- Shipday `Order Delivery Progress` API availability depends on your Shipday plan.

You can start editing the page by modifying `app/page.js`. The page auto-updates as you edit the file.
